# 雨水调蓄计算小工具
## 在线演示

GitHub Pages 访问地址：

https://air-pods.github.io/rainwater-calculator/

用户可以直接通过浏览器打开上述地址使用本工具，无需安装任何软件。
一个使用原生 HTML、CSS 和 JavaScript 编写的单页雨水调蓄计算工具，适用于海绵城市、雨水调蓄、内涝治理和排水方案比选等场景的快速估算与技术说明编写。

## 主要功能

- 单方案雨水调蓄计算
- 汇水面积单位支持 `m²` 和 `ha`
- 自动计算总径流量、径流削减率和等效削减水深
- 自动生成计算过程和工程化计算说明
- 多方案调蓄容积比选
- 自动生成方案比选表格和方案比选说明
- 简洁柱状图展示调蓄容积与径流削减率关系
- 支持复制计算说明、复制表格内容
- 支持导出 CSV
- 支持打印报告或导出 PDF

## 使用方法

1. 直接用浏览器打开 `index.html`。
2. 输入基础参数：
   - 汇水面积 A
   - 径流系数 φ
   - 降雨量 H
   - 调蓄容积 Vs
3. 点击“计算”，查看单方案计算结果。
4. 在“多方案比选”区域输入多个调蓄容积，例如：

   ```text
   500, 1000, 1300, 2000
   ```

5. 点击“生成方案比选及说明”，查看表格、图表和方案比选说明。
6. 根据需要复制说明、复制表格、导出 CSV，或点击“打印报告”导出 PDF。

## 计算公式

总径流量：

```text
Vr = H / 1000 × A × φ
```

其中：

- `Vr` 为总径流量，单位为 `m³`
- `H` 为降雨量，单位为 `mm`
- `A` 为汇水面积，单位为 `m²`
- `φ` 为径流系数

径流削减率：

```text
η = Vs / Vr × 100%
```

其中：

- `η` 为径流削减率，单位为 `%`
- `Vs` 为调蓄容积，单位为 `m³`
- `Vr` 为总径流量，单位为 `m³`

等效削减水深：

```text
Δh = Vs / A × 1000
```

其中：

- `Δh` 为等效削减水深，单位为 `mm`
- `Vs` 为调蓄容积，单位为 `m³`
- `A` 为汇水面积，单位为 `m²`
